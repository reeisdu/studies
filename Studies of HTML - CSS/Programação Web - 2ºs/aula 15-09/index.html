Um seletor é um padrão usado para selecionar os elementos HTML aos quais você deseja aplicar estilos. Os seletores podem ser baseados em elementos, classes, IDs, atributos e relações entre elementos. Propriedade: Uma propriedade é um atributo de estilo que você deseja aplicar a um elemento HTML selecionado. Por exemplo, propriedades podem incluir cores, tamanhos de fonte, margens, preenchimento, etc.

Valor: O valor de uma propriedade especifica como essa propriedade deve ser aplicada ao elemento selecionado.
